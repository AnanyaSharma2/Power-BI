# 📊 Power BI Portfolio

Two end-to-end Power BI projects built from raw data to business-ready dashboards.

---

## 📁 Projects

### 1. [Global Superstore — Profitability Recovery Dashboard](./powerbi-profitability-dashboard/)
> *"Revenue is growing but profit is flat — find out why."*

A 5-page executive dashboard built to diagnose margin erosion across a $2.3M retail operation. Covers discount impact analysis, product profitability, shipping operations, and a live what-if profit simulation.

**Highlights:** 27 DAX measures · Star schema · Time intelligence · What-If scenario model · Dynamic text insights · Decomposition Tree

---

### 2. [Mobile Sales Analysis](./mobile sales analysis/)
> *"Which models, cities, and payment methods drive our sales?"*

A beginner-to-intermediate dashboard analyzing mobile phone sales across 15 Indian cities. Built as the first project in this portfolio to establish core Power BI fundamentals.

**Highlights:** KPI cards · India map visual · Payment method analysis · Monthly trend · Customer ratings breakdown

---

## 🛠️ Skills Across Both Projects

`Power Query` `DAX` `Star Schema` `Time Intelligence` `What-If Parameters` `Conditional Formatting` `Dynamic Text Measures` `Data Modeling` `Power BI Desktop`
